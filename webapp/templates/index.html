DetectX

  • Instructions

DetectX is a platform that utilizes advanced machine learning algorithms to accurately predict the likelihood of a patient having pneumonia. DetectX offers a convenient, reliable, and efficient solution for the early detection and management of pneumonia, helping to improve patient outcomes and support public health efforts.

How to get started

  1. Click on the Diagnose Now button provided in navbar
  2. Select your X-ray image by clicking the select file button

    (Note: The file must be in image(i.e jpeg or jpg) format and not in pdf, word etc)

  3. Submit your file by clicking Submit button
  4. Wait for the response to be generated
